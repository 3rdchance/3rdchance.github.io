2. TMUX

2.1. copy mode

Enter ‘copy mode’ by pressing CTRL+b, [. Use the arrow keys to go to the position from where you want to start copying. Press CTRL+SPACE to start copying. Use arrow keys to go to the end of text you want to copy. Press ALT+w or CTRL+w to copy into Tmux buffer. Press CTRL+b, ] to paste in a possibly different Tmux pane/window. Ctrl-b + : select Ctrl-b - : delete last copyed buffer Ctrl-b # : view a list of copied text

3. Git

3.1. git clone –mirror

git clone –mirror git@coding.jd.com:pmem/pmdk.git git remote update

3.2. merge two repos: merege project-a to project-b

cd path/to/project-b git remote add project-a /path/to/project-a git fetch project-a –tags git merge –allow-unrelated-histories project-a/master # or whichever branch you want to merge git remote remove project-a

3.3. git remote management

git remote add origin git@xxxxurl git branch -M main git push -u origin main
